Approach

  1. Use sklearn.neighbors.KernelDensity to fit a 2-d kernel density estimate to the pairs of, for example, (2030, 2100) in a 2D matrix.
  2. Create a grid of test points. Based on the max/min of each column, create a meshgrid grid_samples.
  3. Use the score_samples method of the kernel density estimate to determine a weight for each test point.
    log_density_values = kde.score_samples(grid_samples)
  4. Convert it to density expo(log_density_values)
  5. Normalize the weights column wise. Divide individual column value by column sum.
  6. Convert back to log values.
  7. Do a filled contour plot as in Gilford20.
